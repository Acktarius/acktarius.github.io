Connect Ledger Live to FutureBit Apollo BTC node

Connect your wallet to your own BTC node, allows you to improve your privacy and independency.

Disclaimer:
this tutorial is delivered “as is” and I deny any and all liability for any damages arising out of using this tutorial.

(Ledger Live being used on a Mac computer connected on the same network as the node, not tried yet on a PC)

1. Downgrade Ledger Live

For some reason, last version of Ledger Live is not working, you have to downgrade to version 2.33.1 , that you can find on repository :

https://github.com/LedgerHQ/ledger-live-desktop/releases/tag/v2.33.1

2. Download SatStack

SatStack bridge your wallet to your node, you will only run it at step #4 of the connection and it might take a while, since it looks for your transactions through the entire blockchain.

https://github.com/LedgerHQ/satstack

3. Modify Bitcoin.conf on your node



Open Terminal
(Leave it open so you can drag « node » folder in the very soon)




Open File Finder



Select Other Locations



Select Computer




Select “opt” folder




Select “apolloapi” folder



Select “backend” folder

you will drag “node" folder in the terminal window after typing “cd"



type “cd ” on the command line and drop the “node” folder path, then valid

Now we will modify bitcoin.conf file, as it is protected, it has to be open with sudo command



type: “sudo nano bitcoin.conf” and valid

note the value of:
rpcuser
rpcpassword
(you’ll need it in Ledger Live to connect)

Add the lines:

rpcbind=127.0.0.1
rpcbind= address IP of your node found in the right tab of the FutureBit page
rpcallowip=127.0.0.1
rpcallowip= adresse IP of the computer from where Ledger Live will be running on the same network
txindex=1

maxconnections= you can increase above 32

Exit “Ctrl + X” and Save by typing “Y”

Valid (return key)



Reboot so the Bitcoin.conf modifications can be taken into account.

Once Node is up and running with decent amount of connections, make sure port 8333, 8332, on your modern are open for the associated IP address.

4. Connect Ledger Live to Node



Open Ledger Live version 2.33.1

click on Experimental Fonctions under Settings


Scroll down and click on connect



Fill with your node parameters obtain from previous step

your node IP address:8332

value of RPC user

value of RPC password

and click Continue



Step #1 : test of connectivity

Step#2 : Select Bitcoin Account



Step#3: have your Ledger nano connected with Bitcoin app open












Step#4 : Run the executable file dowloaded SatStack

BE PATIENT ! don’t do anything until the process is complete, it can take hours !!!

And click on Continue









you will now notice a data symbol beside your Bitcoin account


If you go back to settings you’ll notice the status is now “Ready”

You can now even upgrade to the latest version of LedgerLive, make sure you keep SatStack running in the background.

ET VOILÀ ! you gained some privacy and indepedency !
